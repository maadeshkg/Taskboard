
#  Project Title

TaskBoard - MERN Task Management App

---

#  Description

TaskBoard is a full-stack task management web application built using the MERN stack (MongoDB, Express.js, React, Node.js). It allows users to efficiently manage their daily tasks with features like creating, updating, deleting, filtering, searching, and sorting tasks.

Each task includes important details such as title, description, category, status, and due date. The application ensures proper validation on both frontend and backend to maintain data integrity.

The project demonstrates full CRUD operations with a RESTful API, MongoDB database integration, and a responsive React user interface.

---

#  Key Features

- Create, read, update, and delete tasks (CRUD)
- Filter tasks by status and category
- Search tasks by title
- Sort tasks by due date and status
- Loading spinner for better UX
- Form validation on frontend and backend
- Responsive UI using Bootstrap

---

#  Tech Stack

- Frontend: React (Vite), Axios, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB, Mongoose


## Installation

Backend Setup
```bash
cd backend
npm install

```
Install required packages:
```bash
npm install express mongoose cors dotenv
```
Run backend server
```bash
node server.js
```
Frontend Setup

```bash
cd frontend
npm install
```
Install required packages:
```bash
npm install axios bootstrap
```
Run frontend
```bash
npm run dev
```
